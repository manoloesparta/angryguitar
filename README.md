# Distortion


## Prerequisites

```bash
python 3.7 (or newer)
pip
virtualenv
```


## Setup

Get the clean.wav and distorted.wav files in order to create the dataset

```bash
virtualenv .env -p python3
source .env/bin/activate
pip install -r requirements.txt
```


## Running

```bash
source .env/bin/activate
python main.py
```
